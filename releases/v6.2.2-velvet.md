# v6.2.2-velvet — April 2026

## What's changed

### Refactor: drop fluent-ffmpeg — use native spawn directly

`src/api/transcode.js` and `src/api/waveform.js` now invoke the bundled ffmpeg binary directly via Node's built-in `child_process.spawn` instead of routing through the `fluent-ffmpeg` wrapper library. Behaviour is completely identical — transcoding and waveform generation work exactly as before. `fluent-ffmpeg` has been removed from `package.json`.

### Fix: "Because You Listened To" — max 2 songs per artist

The `songs-by-artists` API endpoint now caps results at **2 songs per artist** after the shuffle. Previously a popular artist with many local tracks could fill all carousel slots, leaving no room for other similar artists. The shelf now shows variety across multiple artists.

### Cleanup: remove obsolete scanner UI controls (LokiJS artefacts)

Ported from upstream `new-sqlite` branch. Two admin panel settings that no longer apply with SQLite have been removed:

- **Save Interval** — not needed; SQLite writes are atomic, there is no periodic flush
- **Pause Between Files** — scan throttle delay; can still be set manually in `default.json` if needed

Removed: UI rows, Vue modal components, API endpoints (`POST /api/v1/admin/db/params/save-interval` and `POST /api/v1/admin/db/params/pause`), and utility functions in `src/util/admin.js`.
