# v6.9.0-velvet — April 2026 — The localized version

`v6.9.0` is the localization milestone release: full runtime language switching across player and admin, stronger locale safety, and centralized language availability controls.

---

## Full i18n rollout (player + admin)

- Player and admin UI strings are now wired to i18n keys instead of hardcoded English text.
- Locale coverage is synchronized across all 12 language files:
  - `en`, `nl`, `de`, `fr`, `es`, `it`, `pt`, `pl`, `ru`, `zh`, `ja`, `ko`
- Dynamic runtime switching updates both static and reactive UI sections without page reload.

---

## Language picker UX refresh

- Language pickers moved from dropdowns to compact one-line flag-button rows.
- Buttons now render real country flag icons (with emoji fallback only if image loading fails).
- Active language state is clearly highlighted in both player and admin.

---

## Safer locale activation

- Locale JSON is validated before activation.
- Invalid or unloadable locale files are rejected.
- The UI remains on the last valid language and shows a visible error notification.
- Picker state is rolled back automatically after failed activation.

---

## Config-driven language availability

New language configuration APIs:

- `GET /api/v1/languages/enabled` (public)
- `GET /api/v1/admin/languages/config` (admin)
- `POST /api/v1/admin/languages/config` (admin)

What this enables:

- Admin controls which languages are visible to users.
- Player and admin pickers now show only enabled languages.
- English is always kept enabled as a safe baseline.

---

## Cross-tab language sync

- Changing language in one open tab (for example admin) now updates other open tabs (for example player) automatically.
- Implemented via `localStorage` change propagation on `mstream-lang`.

---

## Notes

- This release is backward-compatible for existing installs.
- Existing users keep their preferred language via `localStorage['mstream-lang']`.
- Docker users can pin this build with:
  - `ghcr.io/aroundmyroom/mstream-velvet:v6.9.0-velvet`
