# v6.7.2-velvet — CUE Album badge — April 2026

Small cosmetic addition to the Album Library detail view.

---

## feat: "Cue Album" label in album detail header

Albums backed by a CUE sheet (a single FLAC file with embedded cue-point data)
now display a green **Cue Album** label below the Play / Add to Queue buttons in
the detail header. This makes it immediately clear that the track list shows named
cuepoints rather than individual audio files.
