# v6.3.1-velvet — April 2026

Hotfix for three bugs discovered in the v6.3.0 Server Remote (`/server-remote`) immediately after release.

## Fixes

### 404 on page load (`/api/v1/playlist/getall/vpaths-meta`)

The page was calling a URL that never existed. Fixed to use `/api/v1/ping`, which is the real endpoint that returns `{ vpaths: [...], vpathMetaData: {...} }`.

### Auto-DJ adding songs with blank title / artist / album

The `random-songs` API returns a nested shape: `{ filepath, metadata: { title, artist, album, "album-art" } }`. The Auto-DJ code was reading `song.title`, `song.artist` etc. (flat = `undefined`). Fixed to read `song.metadata.title` etc.

### Browse "+" button adding songs with blank metadata, or silently failing

Same nested-shape bug with the `db/metadata` response. Additionally, `resolveAbsPath` in `src/api/server-playback.js` could not find files whose path started with a leading `/` (as returned by the file-explorer API) — the path never matched any vpath prefix so mpv got nothing to play. Both fixed.
