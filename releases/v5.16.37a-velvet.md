# v5.16.37a-velvet — April 2026

## docs: Docker update guide and *_SUBDIR inline examples

### Added
- **"Updating to the latest release"** section added to `docs/docker.md` — explains the `docker compose pull` + `docker compose up -d` workflow for both `latest`-tag and pinned-version users
- Per-type `*_SUBDIR` inline examples added to `compose.yaml`, `docs/docker.md`, and `README.md` — each feature folder block now shows its optional `*_SUBDIR` variable alongside the `ENABLE_*` line, making it clear these are independent and optional

### No functional changes
This release contains documentation improvements only. The server code and Docker image are unchanged from v5.16.37-velvet.

## Upgrade

```shell
docker compose pull
docker compose down
docker compose up -d
```
