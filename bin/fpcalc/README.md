# bin/fpcalc

This directory holds the `fpcalc` binary downloaded automatically at startup by `src/util/fpcalc-bootstrap.js`.

The binary is **not committed to git** — it is downloaded on first use from the official [Chromaprint releases](https://github.com/acoustid/chromaprint/releases).

Supported platforms: Linux x64, Linux arm64, macOS x64, Windows x64.
